2 Get Your API Key

After signing up, create an API key:

  1. Go to API Keys section
  2. Click "Create API Key"
  3. Name it: AI Accountant
  4. Select permission: "Sending access"
  5. Click "Add" and copy the key
Go to API Keys →

3 Paste Your API Key

Enter your API key below (starts with re_):
